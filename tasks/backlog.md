# Backlog

This backlog is intentionally ordered.

Do not start lower-priority work before Phase 0 core loop exists.

## P0 - Required for Minimum Kernel

- Create monorepo skeleton.
- Add shared domain types.
- Add Case Manager.
- Add Evidence Store.
- Add local storage.
- Add Obsidian exporter.
- Add basic report generator.
- Add minimal CLI.
- Add LLM adapter interface.
- Add mock/local model support.
- Add tests for core modules.

## P1 - Useful after Phase 0

- Basic webpage ingestion.
- Playwright ingestion for user-triggered pages.
- Entity extraction.
- Relationship extraction.
- Confidence scoring.
- Hypothesis generation.
- Basic Skill runner.
- Supplier Finder Skill.
- Competitor Snapshot Skill.
- Shopify Snapshot Skill.

## Icebox

These are not for early development:

- Full app marketplace.
- Team enterprise dashboard.
- Paid billing system.
- Large-scale cloud crawling.
- Mobile app.
- Public API.
- MCP server.
- AI Overlay everywhere.

## Rule

Every backlog item must eventually map to one of:

- Memory.
- Evidence.
- Agents.
- Skills.
- Workflows.
- Obsidian.
- User decisions.

If it maps to none of those, remove it or move it to another project.
