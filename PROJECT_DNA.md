# PROJECT DNA

This file defines the identity of Internet Brain OS.

It is the highest-level project file. All future decisions must respect it.

## What we are building

Internet Brain OS is a local-first intelligence layer for the public internet.

It helps users investigate, understand, remember, connect, and act on public information without drowning in tabs, spreadsheets, and disconnected notes.

## What we are not building

We are not building:

- A generic scraper.
- A spam tool.
- A data theft tool.
- A black-hat OSINT platform.
- A tool for bypassing authentication, paywalls, private systems, or access controls.
- A product that locks the user into our cloud.

Scraping is a capability. Intelligence is the product.

## Core mission

Reduce the time and cognitive load required to transform public internet information into evidence-backed decisions.

## Emotional target

When the user first experiences the product, they should feel:

> The internet just became intelligent.

## Product metaphor

Not a scraper.

Not just a browser extension.

Not just a dashboard.

Internet Brain OS is a living second brain for public web intelligence.

## Core principles

1. Every search must make the system smarter.
2. Every conclusion must have evidence.
3. The user owns their data.
4. Local-first beats cloud-first unless cloud is clearly necessary.
5. Obsidian is not just an export target; it is part of the memory strategy.
6. LLMs are replaceable; the Kernel, memory, and evidence model are durable.
7. The product must work with free or low-cost models first.
8. Powerful models should be used surgically for architecture, reasoning, review, and high-value analysis.
9. No feature enters the core unless it strengthens memory, evidence, agents, workflows, or decision-making.
10. The system must stay modular enough that new intelligence domains become Skills, not rewrites.

## The big idea

The user should not ask: "What data can I scrape?"

The user should ask: "What do I want to understand or achieve?"

Internet Brain OS turns that objective into a case, sends agents to work, saves evidence, builds memory, generates hypotheses, and produces useful outputs.

## Permanent north star

We do not want AI to think instead of people.

We want people to never think alone in front of an ocean of information.
