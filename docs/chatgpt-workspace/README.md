# ChatGPT Workspace Setup

This folder prepares Internet Brain OS for use inside a ChatGPT Project/Workspace.

## What this package does

It gives any ChatGPT workspace a stable operating context without depending on one temporary conversation.

## Manual setup

1. Create a new ChatGPT Project named `Internet Brain OS`.
2. Add the repository files listed in `FILE_MANIFEST.md`.
3. Paste the full contents of `PROJECT_INSTRUCTIONS.md` into the Project instructions.
4. Start the first project conversation using `START_HERE.md`.
5. For every new implementation session, use `SESSION_PROMPT.md` and replace the placeholders.

## Source of truth

GitHub remains the canonical source of truth. The ChatGPT Project is a working environment, not a replacement for the repository.

Every meaningful decision must still be committed to the repository and synchronized into institutional memory.
